# Vendor Performance Analysis

A comprehensive data analysis project for evaluating and tracking vendor performance metrics using Python, Jupyter notebooks, and Power BI dashboards.

## 📊 Project Overview

This project analyzes vendor performance data to provide insights into sales trends, performance metrics, and vendor efficiency. It includes multiple analysis notebooks, data processing scripts, and interactive Power BI visualizations.

## 📁 Project Files

### Notebooks

- **Vendor Performance Analysis.ipynb** - Main analysis notebook with detailed vendor performance metrics and visualizations
- **Exploratory Data Analysis.ipynb** - Initial data exploration and data quality assessment

### Scripts

- **ingestion_db.py** - Data ingestion and database operations
- **get_vendor_summary.py** - Vendor summary data generation and reporting

### Data & Reports

- **vendor_sales_summary.csv** - Summary data of vendor sales
- **Vendor Performance Report.pdf** - Detailed performance report
- **vendor_performance.pbix** - Interactive Power BI dashboard

## 🎯 Features

- **Data Processing**: Automated vendor data ingestion and database management
- **Analysis**: Comprehensive performance analysis with statistical insights
- **Visualizations**: Multiple visualizations and dashboards for data exploration
- **Power BI Dashboard**: Interactive `vendor_performance.pbix` for real-time monitoring
- **Reporting**: PDF and CSV reports for stakeholder communication

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook
- Power BI Desktop (for `vendor_performance.pbix`)
- Required Python packages (pandas, numpy, matplotlib, etc.)

### Installation

1. Clone this repository

   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   cd vender-analysis-project
   ```

2. Install dependencies

   ```bash
   pip install pandas numpy matplotlib jupyter
   ```

3. Launch Jupyter Notebook
   ```bash
   jupyter notebook
   ```

## 📈 Usage

### Running Analysis

1. Open `Vendor Performance Analysis.ipynb` for the main analysis
2. Open `Exploratory Data Analysis.ipynb` for initial data exploration
3. Run cells sequentially to regenerate results and visualizations

### Using Python Scripts

```bash
# Data ingestion
python ingestion_db.py

# Generate vendor summary
python get_vendor_summary.py
```

### Power BI Dashboard

1. Open `vendor_performance.pbix` in Power BI Desktop
2. Refresh data connections to latest vendor data
3. Explore interactive visualizations and KPIs
